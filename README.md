# PassBot

PassBot is a collections of bots to send alert by mail, about french passeport,
but not so.

It is build with:
* Python >= 3.11
* Scrapy >= 2.8
* SQLAlchemy >= 2.0
* Alembic >= 1.9
* Pytest >= 7.2
* Docker >= 23
